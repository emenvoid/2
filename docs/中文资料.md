# 关于中文资料

为什么读中文资料
- 一些资料只有中文
- 一些中文资料的质量好过英文
- 同等质量下，读中文的效率更高
- 读不同种类的语言有助于提高阅读能力
  
为什么要尽可能少读
- 知识需要经过实践才能吸收，而时间有限，读得越多就代表实践越少
- 过多的知识不仅派不上用场，还会分散注意力、模糊掉重点
- 大部分资料都是垃圾，找优质资料又过于费时且不一定能找到
- 知识也有边际效应，吸收同类知识的效益会越来越低，到最后仅凭阅读不会有任何提升

怎么读以致用
- 要有自己的目标、目的，然后查找相关的资料并应用到有助于目标的事上
- 如果你不清楚一个资料有什么用，就不要去读，无论它本身有多好，如果你用不上就没有意义
- 避免囤积资料，读了就用，用完就删除或归档，避免文件越下越多，收藏夹越来越臃肿，自己却什么都没学到
- 写作积累真知，用的过程要不断思考并总结，哪对哪错、这么做好再哪不好在哪、该怎么做等等，即使不写给别人看，也要写给未来的自己看

具体步骤
1. 终止被动订阅，学习主动查找(google)
2. 专注于一个领域里相对最优质的信息源
3. 边读边用，先快再慢
4. 费曼学习，解释对于你来说还不够好的地方

# 主动查找
**被动订阅**
普通人总是在被动地接受世界上信息，小时候只知道看教科书，大了只知道刷微博、朋友圈，稍好的则是订阅付费专栏、配置RSS:

[如何看待文中的“大部分的知识付费其实都是大忽悠”？ - 知乎](https://www.zhihu.com/question/63543499)
[在线阅读处理流程：从需求、到方法、再到工具 - 少数派](https://sspai.com/post/36795)

且不论这些被动订阅的知识质量如何、你是否在花钱费时买垃圾，关键的问题在于，你真的需要吗?你现在能用吗?别人永远不知道你需要什么，即使再有良心，也只能尽可能写优质的文章，但再优质的文章，也无法保证是对你有用的。

每一个特立独行者，都应该立即停止这种低效的被动订阅，包括但不限于: 朋友圈、公众号、社交网络、新闻、RSS、博客等，避免漫无目的地上网浏览。

**主动查找**
无论是读书还是上网，首先不是立马开始，而是先搞清楚，自己的目的是什么。如果你没有目的，那么你的目的就是搞清你的目的。

无论你的目的是什么，它都可以用一个词去概括，然后这个词又可以分解成更具体的关键词，普通人称之为知识体系，但本质就是张思维概念图，对于简单的概念甚至不需要思维导图，直接在任意文本软件或纸上进行分解即可。
[你真的会使用思维导图吗？四个方面阐释思维导图的运用 | 人人都是产品经理](http://www.woshipm.com/pmd/919838.html)
[如何建立自己的知识体系？ - 知乎](https://www.zhihu.com/question/19591121/answer/161389318)

然后从某一具体的关键词开始，先写后读，在查找之前先开始写作或者说总结，这样你在写的过程中，文章的大纲、内容就自动帮你搞清了目的，而不用特意地每次都去想"我的目的是什么"。

**Google搜索**
尽管现在google被墙了，但我发现这其实是件好事，反而使得查找优质资料简单了一点，因为那些劣质信息都跑到百度去SEO了，相对而言在google搜到的都确实是真正排名靠前的，而不是别人为了排名靠前专门优化过的。

Google的搜索语法有很多:
[优化网页搜索 - Google 搜索帮助](https://support.google.com/websearch/answer/2466433?hl=zh-Hans)
[google搜索高级命令语法? - 知乎](https://www.zhihu.com/question/21144589/answer/78050196)

真正常用的就3个: 英文引号、减号、site。

**筛选**
首先，最好的筛选方法，就是想一个独特、只有优质资料喜欢用的关键词。

在搜索之前，可以先头脑风暴一下，有哪些相关的关键词，然后一个个试并根据结果及时调整，去除那些容易搜到垃圾的关键词，而对于能够搜到优质信息的关键词，想想还有没有类似的关键词。

每次搜索完要注意积累，留意有哪些通用的优质关键词，比如"原理"、"之道"、"详解"等等。

然后，是珍惜自己的时间、提高自己的过滤标准，不要因为害怕错过就去点击一些垃圾标题，仅仅是因为它可能有用，要时刻警惕各种形式的标题党。
[文章起标题有何技巧？ - 知乎](https://www.zhihu.com/question/19736817)
[我有7招写标题的秘籍，价值300万，但今天免费](https://zhuanlan.zhihu.com/p/20360053)

最后，读到优质资料和劣质资料时都去看看它的标题，对比、分析一下它们的共性与差异，培养自己资料嗅觉，从而在看到标题时第一时间"闻"出它的好坏、优质的可能性、值不值得点开等等。

# 边读边用
任何内容都分为不重要的部分和重点部分，先快速阅读不重要的，再慢速阅读重点。

**快速阅读**
- [快速阅读该怎么自学？ - 知乎](https://www.zhihu.com/question/24716960)
- [快速阅读 (豆瓣)](https://book.douban.com/subject/3916049/)
- [杠杆阅读术 (豆瓣)](https://book.douban.com/subject/4169625/)

这些资料本身就有大量不重要甚至错误的部分，正好可以拿来练习快速阅读。

**慢速阅读**
慢速阅读即理解，建立抽象、未知信息与具体、已知信息的联系
- [怎样深入浅出地回答问题？ - 知乎](https://www.zhihu.com/question/28751773/answer/45548874)

具体做法类似于所谓的主题阅读
- [如何评价《如何阅读一本书》（How to Read a Book）？ - 知乎](https://www.zhihu.com/question/20631409)

我更喜欢时寒冰的解释: 开座谈会
- [读书心得2：如何最快地汲取营养-时寒冰-搜狐博客](http://shihb.blog.sohu.com/155890295.html)

互联网时代，通过书进行组合阅读是很低效的方式，更高效的方法是善用搜索，把某一问题或者概念的相关文章、视频、书的章节等找来，综合、对比、查漏补缺，然后得出自己的理解。

**灵活运用**
在理解一个知识具体是什么之后，要想办法把它和自己的现实生活和理想目标联系起来，想想有哪些可以用得到的地方。
- [李叫兽是如何搜集和整理信息的？ - 知乎](https://www.zhihu.com/question/31154592/answer/51489241)
- [如何拆书？ - 知乎](https://www.zhihu.com/question/21962301)

不要去浪费实践评价知识的好坏对错，而是要去想"我应该怎么用"，不好的知识不代表没用好的知识也不代表有用，要激发自己的绿灯思维，尽可能的去想怎么样才能用好当前的知识。
- [知道做到 (豆瓣)](https://book.douban.com/subject/3208050/)

所有知识当中最有用的就是能够应用到不同领域的底层知识，比如80/20法则，但仅仅靠阅读是不可能理解的，必需学习不同领域的知识，并在不同领域使用，才有可能真正领悟。
- [如何进行跨学科学习？ - 知乎](https://www.zhihu.com/question/38416253)
- [09. 这几年做对的事：系统重写你的底层逻辑](https://zhuanlan.zhihu.com/p/20053878)

**迭代运用**
理论上说我们希望能够像游戏攻略那样，先把一个游戏完全搞懂，然后一步一步完美地通关，但这在现实生活中是不可能的。

一方面，现实中的攻略、教程质量低下，远谈不上完美，另一方面，在你缺乏相应的已知信息时，无法理解更高深、抽象的信息，必须通过实践先积累更多的已知信息，才能够理解那些更深入、进阶的信息，因此需要一边学习一边实践，并根据反馈调整自己的学习。

理想情况下，先学最少且必要的初级知识，然后立马开始实践，理解之后再学更高级一点的最少且必要知识，再实践，这样一直循环到掌握最高深的知识为止。
- [学习一个新领域的知识的最佳方法和最快时间各是什么？ - 知乎](https://www.zhihu.com/question/19550362)

实践时，要像阅读时开座谈会那样，尝试各种各样不同的做法，不要去过早评判哪个更好，然后选择"最好"的一个，只要有用就都尝试一遍才是最好。只有这样，才能对比、分析得出最适合自己的做法，甚至开创新的做法，而不是只会抄袭模仿。

# 费曼学习
**被动笔记**
普通人最喜欢的学习方法就是作笔记，因为不用动脑子，还可以看上去很努力。
- [读书时如何做读书笔记？ - 知乎](https://www.zhihu.com/question/20700854)
- [如何构建自己的笔记系统？ - 知乎](https://www.zhihu.com/question/23427617)
- [什么是康奈尔笔记法？ - 知乎](https://www.zhihu.com/question/19652630)

实际上，计算机时代，资料随时可以查阅，记笔记完全就是浪费时间，反而是一种懒于思考的表现。

然而笔记式学习是如此普遍，以致于当前网上的资料大都是人云亦云，互相抄袭，甚至换个说法就敢说是原创，实际上根本没有自己的理解。

每一个特立独行者，都应该始终避免这种低效的被动学习，包括但不限于: 摘抄、标注、思维导图笔记、大纲笔记、康奈尔笔记、各种笔记软件等。

**主动解释**
比起做一堆不加思考的笔记，真正需要做的是把自己的理解解释给别人，也就是"费曼技巧"。
- [号称终极快速学习法的费曼技巧，究竟是什么样的学习方法？ - 知乎](https://www.zhihu.com/question/20576786)

这个方法的重点，并不是解释给一个5岁小儿，而是借此明确自己的理解具体是什么，假如不去解释，就会造成自以为自己懂了，实际上懂得不够透彻，甚至是错的等等。而通过解释，就可以查漏补缺，不断优化、纠正自己的理解，使之更接近真知。
- [写作，为什么如此重要](https://zhuanlan.zhihu.com/p/19713784)

需要注意的是，写作本身不是重点，是否发表也没有关系，关键在于思考。
- 把自己的想法通过写作缓存下来，从而可以持续地思考。
- 而不是写完一篇文章，发到网上就再也不更新，把自己浅薄的思想解释出来，却不加以反思改进。
- 不断更新、不断深化自己的思想，避免和他人一样浅薄。
